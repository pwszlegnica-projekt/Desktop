## api list
* /login
* /lists (show list lists [get], add list [post])
* /list/{num} (show product list [get], edit list [post], delete list [delete])
* /products (show lists of all products [get])
* /product/{num} (show product detail [get], delete product [delete], edit product [post])
* /list/{num}/product/{num} (show detail about the product on list [get], add the product to list [put], edit product on list [post], remove product from list [delete])
